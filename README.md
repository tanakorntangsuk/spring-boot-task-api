# Spring Boot REST API - Task Management

This project is a RESTful API developed using **Java Spring Boot** for managing personal tasks.
It includes secure **JWT Authentication**, user-based task isolation, and standard backend layered architecture.

## Features
- User registration & login with **JWT**
- Create / Read / Update / Delete **Tasks**
- Each user has their own separate Task data
- Task detail tracking and completion status
- Input validation & global exception handling
- Follows **Controller → Service → Repository** pattern
- Uses **JPA/Hibernate** for database mapping

## Technologies Used
- Java
- Spring Boot
- Spring Security (JWT)
- Spring Data JPA / Hibernate
- MySQL (or any relational DB)
- Maven
